---
title: "Custom App Inbox"
---

In the preceding section, we discussed the configuration for utilizing the default App Inbox implementation.

For developers interested in implementing a custom **Activity** as the **App Inbox** notification center,
Castled SDK offers methods that facilitate the retrieval of messages for display. Additionally, SDK provides essential methods for managing key lifecycle events.

<CodeGroup>

    ```kotlin kotlin
    // Fetching the unread message count directly
    CastledNotifications.getInboxUnreadCount {
      println("unread count is $it")
    }

    // Retrieving all inbox items
    CastledNotifications.getInboxItems { castledInboxItems ->
      println("castledInboxItems $castledInboxItems")
    }

    // Logging read inbox items
    CastledNotifications.logInboxItemsRead(itemsList)

    // Deleting a specific inbox item
    CastledNotifications.deleteInboxItem(inboxItem) { success, message ->
      if (success) {
          println("Deletion was successful: $message")
      } else {
          println("Deletion failed: $message")
      }
    }

    // Logging a click on an inbox item
    CastledNotifications.logInboxItemClicked(inboxItem,"title")
    ```


    ```java java
    // Fetching the unread message count using a callback
    CastledNotifications.getInboxUnreadCount(count -> {
      System.out.println("unread Inbox Count: " + count);
      return null;
    });

    // Retrieving all inbox items
    CastledNotifications.getInboxItems((List<CastledInboxItem> items) -> {
      // Handle the fetched items here
      return null;
    });

    // Logging read inbox items
    CastledNotifications.logInboxItemsRead(itemsList)

    // Deleting a specific inbox item
    CastledNotifications.deleteInboxItem(inboxItem, new kotlin.jvm.functions.Function2<Boolean, String, Unit>() {
      @Override
      public Unit invoke(Boolean success, String message) {
          // Handle the success and message here
          if (success) {
              System.out.println("Item deleted: " + message);
          } else {
              System.out.println("Error: " + message);
          }
          return null;
      }
    });

    // Logging a click on an inbox item
    CastledNotifications.logInboxItemClicked(inboxItem,"title");

    ```
