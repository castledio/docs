### Edit Physical Address

The physical addresss created can be modified if required. A physical address can be modified only if it is not being used in any running campaigns or workflows.

![](/images/physical-address-6.png)

### Delete Physical Address

An existing physical address can be deleted only if it is not being used in any active running campaigns or workflows.

![](/images/physical-address-7.png)
