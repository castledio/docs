---
title: "SQL Builder"
---

Castled provides a SQL segment builder to enable the marketers or data teams to create custom segments tailor made for running the most hyper personalised campaigns. This options gives more flexibility to the marketer who has SQL knowledge and are aware of the entity details in the data warehouse.


### 1. Select the SQL Builder option while creating the Segment.

![](/images/sql-builder-1.png)

### 2. Enter the SQL Query.  

Select the User object to be segmented and type in the SQL query for creating the required User Segment.  

![](/images/sql-builder-2.png)

### 3. Preview the SQL query.

Preview the SQL query to verify the results and complete the segment creation process by clicking Next.

![](/images/sql-builder-3.png)
