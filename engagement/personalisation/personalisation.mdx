---
title: "Personalisation"
---

Castled provides the option to personalise the campaign content being created. Personalisation is available while creating content in all the supported channels.  

The UI controls which supports personalisation will have the ghost text **Type @ to personalise**.  

### Type @ to personalise.  

This opens the personalisation popup where the following details needs to be captured.  
- User attribute to be used for personalisation.  
- What should happen in scenarios where the user attribute selected for personsalisation has no value associated.  
![](/images/personalisation-1.png)

In cases where the user attribute value is null one of the following option needs to be selected.  
- Do not proceed with the campaign for that user. 
- Proceed with campaign with a default value captured.

![](/images/personalisation-2.png)
