---
title: "Create Template"
---

Email templates are to be predefined in Castled to be used in Email campaigns or Email steps in workflow. Defining templates enables the marketer to reuse an existing template in multiple campaigns with a little or no change in the template. 

### 1. Go to List Templates page and click CREATE TEMPLATE button.

![](/images/email-template-1.png)

### 2. Choose the User segment to be used.

User segment needs to be captured if personalisation is required in the email template being created. Make sure the User Segment is already created before the email template is created.

![](/images/email-template-2.png)

![](/images/email-template-3.png)

### 3. Choose the template type.

Castled supports three template types : 
1. Drag & Drop Editor 
2. HTML Editor 
3. Rich text Editor

![](/images/email-template-4.png)

### Drag & Drop editor
![](/images/email-template-7.png)

### HTML Editor
![](/images/email-template-5.png)

### Rich Text Editor
![](/images/email-template-6.png)

### 4. Personalise the email template.  

Castled provides the option to personalise the Email template being created. Type **@** to invoke the personalisation popup.Refer [personalisation](/engagement/personalisation/personalisation) section for more details.

![](/images/email-template-8.png)

### 5. Select the User property required for personalisation.  

Select the User property to be used in the template and capture the default value for cases where the property value might not be available for that User.

![](/images/email-template-9.png)

### 6. Complete the template design. 
![](/images/email-template-11.png)

### 7. Send test email.
![](/images/email-template-12.png)

### 8. Capture the test email subject, from address and recipient email address to whom the test email needs to be send.
![](/images/email-template-14.png)

### 9. Check the recipient email address to verify the email template.  
![](/images/email-template-15.png)

### 10. Save the email template so that it is listed in the List Template screen. 
![](/images/email-template-16.png)
