---
title: "Event Schema"
---

Data teams needs to define the Event schema to be used in Customer Segmentation.
Go to Schema > Events in the menu and follow the following steps to create an
Event Schema in Castled.

## 1. Click on Create Event schema

![](/images/event-schema-1.png)

## 2. Select the warehouse config and enter the query to define the event entity being created.

![](/images/event-schema-2.png)

## 3. Run the query and preview the result.

![](/images/event-schema-3.png)

## 4. After previewing the query result click Next.

![](/images/event-schema-4.png)

## 5. Capture the schema name and map the Event timestamp field.

![](/images/event-schema-5.png)

## 6. Capture the User object associated with the event using the Relationship section.

![](/images/event-schema-6.png) ![](/images/event-schema-7.png)
![](/images/event-schema-8.png)

## 7. Save and finish to create an Event schema.

![](/images/event-schema-9.png)

<Note>An Event Schema can be associated with only one User Schema</Note>
